## 容器查询实现
1. 使用容器查询令容器高度超过定值的时候，右下角显示一个贴条；

2. 使用-webkit-box使文字显示2行，且调大省略号的字体大小，使右下角空出一块；

3. 合并上面两个元素，并隐藏溢出部分。
